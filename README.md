## Overview

Single-page scrolling website built as the presentation for my maturità colloquio orale (2026). The page itself is a demonstration of the Informatica skills it describes — vanilla HTML, CSS and JavaScript, no frameworks, deployed on GitHub Pages.

**Overarching theme:** L'intelligenza artificiale, dal progetto alla rete.

- **Vanilla HTML / CSS / JavaScript** — no frameworks, no build step
- **GitHub Pages** — deployed directly from this repo

- Interactive AI Act risk pyramid
